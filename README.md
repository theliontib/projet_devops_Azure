# Projet DevOps Azure

## Objectif
Apprendre et mettre en place une architecture DevOps avec Docker, Azure et CI/CD.

---

## Stack
- Docker
- Azure
- GitHub Actions
- Terraform (à venir)
- Kubernetes (bonus)

---

## Fonctionnement

1. Création d'une application simple
2. Dockerisation
3. Push vers Azure Container Registry
4. Déploiement sur Azure

---

## CI/CD

Pipeline automatique avec GitHub Actions :
- Build Docker
- Push image
- Déploiement

---

## Auteur
Projet réalisé dans le cadre d’une auto formation DevOps. Guidé par l'IA sur une suite de TP.
